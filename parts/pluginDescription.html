auto-yapi

功能: easy yapi插件扩展插件，支持自动导出能力。

  • git push事件自动推送更改接口导yapi接口管理平台
  • 定时任务自动推送文档导接口管理平台

依赖自扩展的的EasyApi，不可自动升级

配置信息：
    在File->settings->OtherSettings->AutoYapi
  • 监听配置中导出目录：指定需要导出生成yapi文档的目录（一般为interface父目录，只有此目录下的文件才会导出生成文档）若更改的文件或者文件的引用在这个目录下，将会对该文件进行yapi文档的替换更新。
  • Git Push配置中设置可以导出yapi文档的监听目录，（一般为接口所在的module目录）该目录会被监听，在push操作触发时，指定目录下的文件出现变动才会触发导出动作
  • 定时任务配置可以设置定时导出的首次执行时间以及时间间隔，该操作会定时将导出目录下的所有文件导出yapi文档（saveOrUpdate）。
